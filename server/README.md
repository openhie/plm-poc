# Patient Level Monitoring - Proof of Concept

## Project setup
```
npm install
```

### To run tests
```
npm test
```

### To start server
```
npm start
```

Then browse to port 3000 on the server, e.g:
http://localhost:3000/

To change the port set the PORT environment variable before starting, e.g.:
```
PORT=8080 npm start
```
